# Flutter Dashboard Application

A modern and responsive **Flutter Dashboard App** built to showcase clean UI design, reusable widgets, and structured layout. This project demonstrates dashboard-style screens commonly used in admin panels, analytics apps, and business applications.

---

# Features

✅ Clean and modern dashboard UI  
✅ Responsive layout (works on different screen sizes)  
✅ Reusable widgets & modular code structure  
✅ Dashboard cards / stats sections layout  
✅ Smooth UI using Flutter Material Components  

---

# Tech Stack

- Flutter
- Dart
- Material UI
